# 租房APP功能快速参考 🚀

## 🏠 系统概览

完整的租房管理系统，包含订单管理和钱包系统。

---

## 📋 订单管理系统

**访问地址：** http://localhost:5174/my/orders

### 核心功能
- ✅ 创建订单（服务详情页提交预约）
- ✅ 查看订单（支持状态筛选）
- ✅ 修改订单状态（5种状态）
- ✅ 删除已取消订单（单个/批量）

### 订单状态流程
```
待确认 → 已确认 → 服务中 → 已完成
  ↓        ↓
  → 已取消 → 删除
```

### 快捷操作
| 状态 | 可用操作 |
|-----|---------|
| 待确认 | 确认订单、取消订单 |
| 已确认 | 开始服务、取消订单 |
| 服务中 | 确认完成 |
| 已完成 | 再次预约 |
| 已取消 | 删除订单 |

## 💰 钱包系统

**访问地址：** http://localhost:5174/my/wallet

### 核心功能
- 💰 账户余额（充值/提现）
- 🏆 谷粒点积分（签到/兑换）
- 📝 交易记录（分类查询）

### 快速操作
| 功能 | 说明 |
|-----|------|
| 充值 | 最低¥10，充值100元送10点 |
| 提现 | 1-3工作日到账 |
| 签到 | 每日赚5-10谷粒点 |
| 兑换 | 100谷粒点 = ¥10 |
