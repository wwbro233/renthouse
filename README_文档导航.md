## ❓ 常见问题

### Q1: 如何快速理解整个系统？
**A:** 先看[功能概述](个人中心功能概述.md)，再根据需要深入[技术文档](个人中心功能实现技术文档.md)的具体章节。

### Q2: 代码在哪里？
**A:** 
- 页面组件：`src/views/My*.vue`
- 状态管理：`src/stores/*.js`
- 路由配置：`src/router/index.js`

### Q3: 如何调试？
**A:** 
1. 打开浏览器开发者工具
2. 使用Vue DevTools插件
3. 查看localStorage数据
4. 阅读控制台日志

### Q4: 如何集成后端？
**A:** 参考[技术文档](个人中心功能实现技术文档.md)的"未来优化方向-后端集成"章节。

### Q5: 数据存储在哪里？
**A:** 
- 当前：localStorage（键值：link-house-*）
- 生产：需要后端API + 数据库
